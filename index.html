Introduction

What is Javascript? Javascript was initially created to "make web pages alive". The programs in this language are called scripts

They can be written right in a web page's HTML and run automatically as the page loads

Scripts are provided and executed as plain text. They don't need specialist preparation or compilation to run.

Interesting Fact about JavaScript

When Javascript was created, it initially had another name: "LiveScript". But Java was very popular at that time, so it was decided that positioning a new language as a "younger brother" of Java would help. However Javascript has "grown up" and no longer has any relation to Java at all.

There are three great things things about Javascript

  • Full integration with HTML/CSS
  • Simple things can be done simply
  • Support all major browsers and enabled by default
Javascript is the only browser technology that combines these three things.

Hello, World!

Javascript programs can be inserted almost anywhere into inline an HTML document using the <script> tag

<script>

alert( 'Hello, world!' );

</script>

This calls an alert on the browser. The browser will automatically play this. However do not do this.

Rather Javascript is linked similarally to how css style sheets .css are linked. However this is for Javascript code is linked with a .js file.

<script src ="script_path.js"></script>

If you want to attach several scripts, use multiple tags:

<script src ="script_path1.js"></script>

<script src ="script_path2.js"></script>

<script src ="script_path3.js"></script>

This reduces traffic and makes page load faster.

Variables

Variables are "storage containers" for data in your code. Until recently there was only one way to create a variable in Javascript. the var statement. But in the newest Javascript versions we have two more ways. let and const

We can use these variables to store goodies, visitors and other data.

The statement below creates (in other words: declares) a variable for the name "message

let message

Now, we can put some data into it by using the assignment operator =

Let message;

message = 'Hello'; // store the string

The string is now saved into the memory associated with the variable.

Let message;

message = 'Hello'; // store the string

alert(message); // shows the variable content

To be concise, we can combine the variable declation into a single line.

Let message = 'Hello'; // store the string

alert(message); // shows the variable content

With multiline variants use

let user = 'John';

age = 25;

message = 'Hello';

Variable Naming

There are two limitations on variable names in Javascript

  1. The name must contain only letters, digits, or the symbols $ and _

  2. The first character must not be a digit

Examples of valid names:

  • let userName;
  • let test123;

When a name contains multiple words, camelCase is commonly used. The first word is lowercase and every word after it has a Capital letter for the first letter

Numbers

Numbers are either with decimals or without demicals. Extra large or extra small numbers can be written with exponent notations. Integers (numbers without a period or exponent notation) are accurate up to 15 digits. The maximum number of decimals is 17.

Never write a number with a leading zero (like 07)
